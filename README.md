# Personal portfolio site
This repository hosts a GitHub Pages site and provides a Docker‑based local development environment so you can preview changes before deploying.
Using Docker ensures a consistent setup across machines—no need to install Ruby, Bundler, or Jekyll directly on your system.

## Prerequisites
The project uses:

* Docker
* Docker Compose
* Git
* Node
* NPM

## Local Development
Start the development server by running:
```bash
docker compose build && docker compose up
```
Then open your browser to: [http://localhost:4000]()
